## Description

LedgerLens is the frontend for an AI-assisted bank statement processing app, built for uploading, tracking, and reviewing parsed PDF statements with real-time status updates.

Built with: [Next.js](https://nextjs.org/)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
